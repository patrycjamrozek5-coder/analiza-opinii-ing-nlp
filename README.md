# Analiza opinii klientów ING z wykorzystaniem NLP

Projekt przedstawia analizę opinii klientów ING z wykorzystaniem metod przetwarzania języka naturalnego oraz modeli uczenia maszynowego.

## Cel projektu

Celem projektu było przeanalizowanie opinii klientów banku ING, określenie ich sentymentu oraz sprawdzenie, które metody reprezentacji tekstu najlepiej sprawdzają się w klasyfikacji opinii.

## Zakres projektu

W projekcie wykonano:

- zebranie i przygotowanie danych tekstowych,
- czyszczenie opinii klientów,
- eksploracyjną analizę danych,
- analizę rozkładu ocen i sentymentu,
- przygotowanie danych do modelowania,
- klasyfikację opinii z wykorzystaniem metod NLP,
- porównanie wyników modeli,
- interpretację wyników i wnioski końcowe.

## Wykorzystane technologie

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook
- NLP
- Bag of Words
- TF-IDF
- Logistic Regression
- Word2Vec

## Modele i metody

W projekcie wykorzystano między innymi:

- Bag of Words + regresja logistyczna,
- TF-IDF + regresja logistyczna,
- Word2Vec jako metoda reprezentacji tekstu,
- macierze pomyłek,
- accuracy, precision, recall i F1-score.

## Wnioski

Projekt pokazuje pełny proces analizy danych tekstowych: od przygotowania danych, przez eksplorację, aż po budowę i ocenę modeli klasyfikacji sentymentu.
